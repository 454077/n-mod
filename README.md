# n-mod
<h1>A fan-modded version of Mr. LandGreen's <a href="https://landgreen.githb.io/sidescroller">n-gon</a>.</h1>
<img src="favicon.svg" width="250px"></img><hr>
I work on this project a lot in my spare time.
It includes portions of the following games and mods:
<ul>
  <li><a href="https://github.com/landgreen/n-gon">n-gon (primary directory) by Landgreen</a></li>
  <li>guns, field, and tech from <a href="https://github.com/Whyisthisnotavalable/n-scythe">n-scythe by WhyIsThisNotAvailable</a></li>
  <li><a href="https://github.com/c-rxxp-y/n-gon-enhanced">n-hanced by c-rxxp-y</a></li>
  <li><a href="https://github.com/3xionDev/n-gon-upgraded">n-upgraded by 3xionDev</a></li>
  <li>maps from <a href="https://coaldeficit.github.io/c-gon">c-gon by coalDeficit</li>
</ul>

I have made some changes:
<ul>
  <li><em>Lorentz Transformation</em> stacks to 3×, and effects stack properly</li>
  <li>added <em>missile-bot</em> and <em>plasma-bot</em> upgrades</li>
  <li>removed certain maps that either had bugs related to testing, now only accecible via warp powerUp</li>
  <li>Changed testing keybinds:
    <ul>
      <li>` spawns coupling</li>
      <li>Shift + ` spawns boost</li>
      <li>3-9 and 0 spawn research, gun, field, tech, difficulty, instructions, warp, and entanglement respectively</li>
      <li>Shift + 8, 9, and 0 spawn block, mob, and boss respectively</li>
    </ul></li>
  <li>powerUp-type bosses are each 1 tier higher, and are insta-deleted on level.final, "taunting" finalBoss when doing so</li>
  <li><em>Working Mass</em> stacks to 3×</li>
  <li><em>Marginal Utility</em> works on <strong>scythe</strong> with <em>titanium nitride</em></li>
  <li>added sound effect system, <em>sounds</em> JUNK tech</li>
  <li>Warp option at top of <em>warp</em> menu advances to next level</li>
  <li>finalBoss is now immune to mob clearing via testing keybind</li>
  <li>added a "removed levels" list; the levels therein can now <strong>ONLY</strong> be accessed in-game via warp powerup</li>
  <li>Command Console (requires testing)</li>
  <li><em>swap meet</em> properly stacks</li>
  <li>Added back <em>microtransactions</em></li>
</ul>
